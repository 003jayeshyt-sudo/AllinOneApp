## Project Overview

All-In-One Smart App is an Android application developed as a final year academic project.  
The main purpose of this app is to provide access to multiple popular platforms such as **Social Media, Shopping, Food, and Entertainment** inside a single Android application using **WebView**.

Instead of installing many different apps, users can use this one app to access everything in one place. This helps in saving storage space and makes mobile usage more convenient.

---

## Objectives

- To reduce mobile storage usage  
- To combine multiple apps into one application  
- To provide easy and fast access to popular platforms  
- To design a simple and user-friendly interface  

---

## Features

- Category-based tabs (Social, Shopping, Food, Entertainment)
- Card-based app tiles
- WebView integration
- Clean and simple UI
- Lightweight Android application
- Signed APK generated

---

## Technologies Used

- Programming Language: Kotlin  
- IDE: Android Studio  
- UI Design: XML  
- Core Component: WebView  
- Build Tool: Gradle  
- Version Control: Git & GitHub  

---

## Project Structure

- `app/` – Main application module  
- `res/layout/` – XML layout files  
- `res/drawable/` – App icons and images  
- `MainActivity.kt` – Main activity file  
- `SplashActivity.kt` – Splash screen activity  

---

## How to Run the Project

1. Clone the repository  
git clone https://github.com/003jayeshyt-sudo/AllInOneApp.git
2. Open Android Studio  
3. Select **Open Existing Project**  
4. Choose the project folder  
5. Let Gradle sync complete  
6. Run the app on emulator or real device
